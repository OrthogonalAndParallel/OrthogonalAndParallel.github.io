系统设置

  • 允许任何来源 sudo spctl --master-disable
  • 启动台图标
    • 横轴 defaults write com.apple.dock springboard-rows -int 7
    • 纵轴 defaults write com.apple.dock springboard-columns -int 11
    • 生效 defaults write com.apple.dock ResetLaunchPad -bool TRUE;killall Dock
  • 环境变量
    • 编辑 vim ~/.base_profile
    • 设置
    export M2_HOME=/Users/wings/Documents/apache-maven-3.6.2
export NODE_GLOBAL=/usr/local/node_global
export PATH=$PATH:$M2_HOME/bin:$NODE_GLOBAL/bin
    • 生效 source ~/.base_profile
